A web-based digital companion for early mental wellness support among youth and young adults.

## Quick Start

1. Run setup: `./scripts/setup.sh`
2. Start backend: `cd backend && python manage.py runserver`
3. Start frontend: `cd frontend && npm start`

## Project Structure

- `backend/` - Django API and database
- `frontend/` - React website
- `ai/` - Sentiment analysis and chatbot logic
- `docs/` - Documentation and guides
- `tests/` - Automated tests
